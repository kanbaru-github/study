## アロー関数
- ES6から追加
### 関数を短くかける
- 一行で書くこともできる({}が不要)(2行で書く場合は必要)
```js
const hello = name =>console.log('Hello,'+name);
hello('kanbaru');
```
### thisを束縛しない
- 前提：thisは原則コードを上に遡って一番最初に火つけたオブジェクトを参照する
- アロー関数で書かない場合は上記のケースに当てはまる
- アロー関数で書く場合は呼び出す関数を一番最初の関数に固定する
